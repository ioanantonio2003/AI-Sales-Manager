## üìå Despre Proiect

**AI Sales Manager** este o aplicatie moderna care elimina bariera tehnica dintre utilizatorii de business si bazele de date. In loc sa scrie cod SQL manual, un manager poate pur si simplu sa intrebe: *"Care sunt cele mai vandute produse?"* sau *"Cati clienti noi avem luna aceasta?"*.

Sistemul foloseste cel mai nou model **Google Gemini 2.5 Flash** pentru a intelege structura bazei de date si pentru a genera interogari SQL precise in timp real.

### Functionalitati Cheie
* **Text-to-SQL:** Conversie automata a intrebarilor din romana/engleza in cod SQL executabil.
* **Analiza Datelor:** Extrage informatii despre stocuri, vanzari si clienti direct din baza de date.
* **Interfata Vizuala:** Dashboard interactiv construit cu **Streamlit**.
* **Siguranta:** Agentul este configurat sa nu modifice datele , prevenind stergerea accidentala.
